## [0.1.0] - 2026-02-13

### Added
- Initial project setup
- Basic roguelike terminal game framework
- Player character with HP and sanity system
- Fog of war mechanics
- Sound system with directional audio cues
- Game entities and movement system
- Effect system for status effects
